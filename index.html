GitList


an elegant and modern git repository viewer

‹ ›

GitList allows you to browse repositories using your favorite browser, viewing files under different revisions, commit history and diffs. GitList is free and open source software, written in PHP, on top of Silex and the Twig template engine.

Development

GitList is actively maintained by many developers around the world. You can fork the project on GitHub and start contributing yourself. The project is continuously integrated, so we can improve code quality and build effortlessly.
